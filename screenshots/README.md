# Screenshots placeholder

В эту папку можно добавить скриншоты:
- form.png — скриншот формы заявки
- admin.png — скриншот страницы /admin

Для добавления скриншота используйте команду:
```bash
generate_image --prompt "A clean lead capture form with name, email, phone, company fields" --output leadkeeper/screenshots/form.png
```
